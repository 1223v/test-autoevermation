# Changelog

All notable changes to "Test-AutoEvermation" will be documented in this file.

## [1.0.0] - 2024-12-20

### Added
- Initial release
- Generate Unit Test command (Ctrl+Shift+T)
- Analyze Java Code command (Ctrl+Shift+A)
- Sidebar UI for settings and quick actions
- API server connection management
- Support for JUnit 4/5 and Mockito/EasyMock
- Automatic test file path resolution (Maven/Gradle conventions)
- Dependency analysis for test generation
- Status bar indicator
